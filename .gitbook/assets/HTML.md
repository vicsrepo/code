# HTML

HTML (HyperText Markup Language) je základní programovací jazyk používaný k vytváření webových stránek a webových aplikací. Je to označovací jazyk, který používá sadu značek (tags) k strukturování obsahu na webové stránce. Zde je stručné shrnutí:

1. **Struktura a značky**:
   - HTML dokumenty začínají deklarací `<!DOCTYPE html>`, následuje značka `<html>`, která obsahuje celý obsah webové stránky.
   - Hlavní části HTML dokumentu jsou `<head>` a `<body>`. `<head>` obsahuje metadata, odkazy na stylové listy a skripty, zatímco `<body>` obsahuje skutečný obsah stránky.

2. **Základní značky**:
   - `<h1> až <h6>`: Nadpisy různých úrovní.
   - `<p>`: Odstavec.
   - `<a>`: Hypertextový odkaz.
   - `<img>`: Obrázek.
   - `<div>` a `<span>`: Kontejnerové značky pro skupinování a stylování obsahu.

3. **Atributy**:
   - Značky mohou mít atributy, které poskytují další informace. Například `<a href="url">` definuje odkaz, který vede na specifikovanou URL.

4. **Formátování**:
   - HTML podporuje základní formátování textu, jako je tučné písmo (`<b>` nebo `<strong>`), kurzíva (`<i>` nebo `<em>`), seznamy (`<ul>`, `<ol>`, `<li>`) a další.

5. **Formuláře**:
   - HTML umožňuje vytváření interaktivních formulářů pomocí značek jako `<form>`, `<input>`, `<textarea>`, `<select>`, a `<button>`.

6. **Multimedia**:
   - HTML podporuje vkládání multimediálního obsahu, jako je zvuk (`<audio>`), video (`<video>`) a integrované objekty (`<embed>`, `<object>`).

7. **HTML5**:
   - Nejnovější verze HTML, HTML5, přidává nové funkce a značky, jako jsou `<article>`, `<section>`, `<header>`, `<footer>`, a `<nav>`, které zlepšují sémantiku a strukturu dokumentů. Rovněž zahrnuje nativní podporu pro audio a video, grafiku pomocí `<canvas>` a mnoho dalších pokročilých funkcí.

HTML je základním stavebním kamenem webového vývoje a je jednoduché na naučení, což z něj činí ideální výchozí bod pro každého, kdo se chce stát webovým vývojářem.
